
* Instalando git e git bash
* criando uma conta no Git Hub

* configurando o Git com suas credenciais
-> git config --global user.name seu nome
-> git config --global user.email seu email

* Criar um repositório no Git Hub
* Criar um repositório Local

* inicializar o repositório Local vfdgg
-> git init (dentro do rep local)

* conectando o repositório local ao remoto
- copia o https:// do remoto
-> git remote add (apelido origin) https:// link copiado

* verificando
-> git remote -v

* Ou usando o Git Clone
-> git clone http copiada .

* verificando o estado do status
-> git status

* adicionar arquivos ao índice
-> git add (nome do arquivo + extensão)
-> git add * (para adicionar todos)

* comitando para confirmar as alterações no header
-> git commit -m "frase do comit"

* enviando para o repositório remoto
-> git push origin master
